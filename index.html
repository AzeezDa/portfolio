Some Programming Projects

Eqsolver

A Numerical Equation Solver Library for Rust. This library includes solvers for non-linear system of equations using e.g. Newton-Raphson's method and Gauss-Newton's algorithm. I also includes a solver for systems of ordinary differential equations.

Ceetle

A Computational Tree Logic (CTL) Semantic Verifier. This is a library that provides the ability to define CTL models and verify the validity of formulas on these models. Most of the verification algorithms use modified version of a Breadth-First Search on graphs.

JavaScriptOne

A Small Collection of Small JavaScript Projects. I use this repository to put small JavaScript projects and experiments such as simple colour games or a piano with a few key. Check out the deployment in this link.

CSVGrapher

A Python-based GUI application built with Qt5, Plotly and Pandas to visualize CSV files. I built this application after a request for a tool that helps visualize data in CSV files. It allows the user to query data and plot it as pie, line or bar charts with different customizations.

Planets

A C++ simulation of Newton's Law of Gravitation. This is a repository that includes code I used to visualize Newton's Law of Gravitation using SFML for graphics. The results can be seen in this video on my YouTube channel

IS1500 Project

A game built on an Uno32 w/ Basic I/O Shield. This was a project of the IS1500 Computer Organization and Components course at KTH Royal Insitute of Technology. This project was made together with Herdi2.

NEOPHOEBE

A Rust simulation of Disease Spread with a Python-based interface. This is a command-line interfaced program that takes a file of parameters and simulates a disease spread based on those parameters then outputs the result as a .csv file, a figure or both.
